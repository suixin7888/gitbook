# 函数定义和调用 {#函数定义和调用}

## &lt;1&gt;定义函数 {#定义函数}

定义函数的格式如下：

```py
    def 函数名():
        代码
```

demo:

```py
# 定义一个函数，能够完成打印信息的功能
def printInfo():
    print '------------------------------------'
    print '         人生苦短，我用Python'
    print '------------------------------------'
```

## &lt;2&gt;调用函数 {#调用函数}

定义了函数之后，就相当于有了一个具有某些功能的代码，想要让这些代码能够执行，需要调用它

调用函数很简单的，通过**函数名\(\)**即可完成调用

demo:

```py
# 定义完函数后，函数是不会自动执行的，需要调用它才可以
printInfo()
```
